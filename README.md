# Edy Rodriguez Portfolio

## Description

Welcome to my multilingual web portfolio, designed to showcase my experience as a web developer along with a selection of projects I have worked on. The portfolio is available in English, Spanish, and French. Feel free to explore my work and get in touch if you have any questions or opportunities!

## Installation Instructions

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/edy-rodriguez-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd edy-rodriguez-portfolio
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Usage

Once the development server is running, you can open the portfolio in your web browser and navigate through the different sections to see my experience and projects. Clicking on a project will reveal more detailed information about it. You can also use the contact form to get in touch with me directly.
